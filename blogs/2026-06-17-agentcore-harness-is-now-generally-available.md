The managed agent harness in Amazon Bedrock AgentCore is now generally available, providing the orchestration loop, tool execution, context management, state persistence, failure recovery, and session isolation that an agent needs. Customers define an agent in configuration (model, tools, skills, instructions) and AgentCore runs it in an isolated environment, decoupling the harness from the model so providers can be switched mid-session, with a single CLI command to export to Strands-based code. Available in all Commercial Regions where AgentCore runs.
