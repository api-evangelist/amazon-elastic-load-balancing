Amazon MQ for RabbitMQ now supports private networking, letting brokers reach private resources in your VPC such as LDAP and OAuth 2.0 identity providers or self-hosted RabbitMQ brokers without public exposure. It uses Amazon VPC Lattice, AWS Resource Access Manager, and AWS PrivateLink, replacing prior Network Load Balancer and NAT Gateway workarounds. Available in all Regions where Amazon VPC Lattice is offered.
