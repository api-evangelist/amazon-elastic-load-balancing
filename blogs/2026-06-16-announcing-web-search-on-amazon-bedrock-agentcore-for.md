Web Search on Amazon Bedrock AgentCore is now generally available, a fully managed tool that grounds agent responses in current web knowledge with zero data egress, keeping data residency within your AWS environment. Built on Amazon's search infrastructure used by Alexa+, Amazon Q Business, and Kiro, it combines a web index with knowledge graph data and is exposed as a built-in connector target on the AgentCore gateway via the Model Context Protocol. Available in US East (N. Virginia).
