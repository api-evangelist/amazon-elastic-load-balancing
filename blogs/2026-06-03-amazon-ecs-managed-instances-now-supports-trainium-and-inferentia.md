Amazon ECS Managed Instances now supports AWS Trainium and AWS Inferentia, purpose-built AI accelerators designed to deliver scalable performance and cost efficiency for training and inference. Users can specify accelerator instance types including Inferentia2, Trainium1, and Trainium2, and add NEURON_CORE=all configuration to task definitions for optimal performance with full EC2 capabilities.
