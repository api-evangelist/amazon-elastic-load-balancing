AWS Transform for mainframe now delivers a connected, traceable reimagine experience from assessment through code generation for z/OS COBOL and PL/I workloads. A portfolio assessment catalogs discrete business functions that flow into the reimagine workflow, generating development-ready requirements with full traceability into Kiro and other IDEs through MCP-based integrations, so every requirement traces back to source code. Available in all Regions where AWS Transform for mainframe runs.
