AWS Glue Data Catalog now lets you enrich tables, including S3 Tables, with glossary terms and custom metadata fields, then use a new Glue Search API to find data by semantic meaning alongside technical metadata. Any MCP-compatible agent including Claude Code, Kiro, Cursor, and Codex can use it via the aws-data-analytics plugin from the Agent Toolkit for AWS. Available in preview in US East (N. Virginia, Ohio), US West (Oregon), and Europe (Ireland).
