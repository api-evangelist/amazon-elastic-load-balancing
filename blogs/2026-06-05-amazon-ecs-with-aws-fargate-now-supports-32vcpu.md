Amazon ECS with AWS Fargate now supports 32vCPU compute configurations for both x86-based and ARM-based Linux workloads, with memory options of 60 GiB, 120 GiB, or 244 GiB. These configurations enable high-performance computing, large-scale data processing, AI inference, and other compute-intensive workloads, and are available on both Fargate and Fargate Spot capacity providers across all AWS commercial and GovCloud regions.
