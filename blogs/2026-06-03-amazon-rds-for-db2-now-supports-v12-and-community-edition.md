Amazon RDS for Db2 now supports IBM Db2 v12.1, and the new Db2 Community Edition provides all features available in Standard and Advanced Editions with no commercial software licensing charges for development and test applications. This enables developers to build and test Db2 applications on Amazon RDS without licensing costs before deploying to production.
