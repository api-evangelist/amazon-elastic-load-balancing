Amazon Bedrock AgentCore Runtime now supports interactive shells through a new InvokeAgentRuntimeCommandShell API, providing persistent terminal access into agent sessions over WebSocket. Developers can authenticate and interact with coding agents like Claude Code directly within isolated microVMs, with features including automatic reconnection and support for up to 10 concurrent shells per runtime.
