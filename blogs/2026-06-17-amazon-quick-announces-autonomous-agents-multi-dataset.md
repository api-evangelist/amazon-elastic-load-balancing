Amazon Quick adds autonomous agents that handle recurring tasks at granular autonomy levels, multi-dataset analytics that query across sources including Snowflake and relational databases via natural language, and a redesigned conversational activity feed. Quick inherits semantic intelligence from catalogs like AWS Glue, Databricks Unity Catalog, and Collibra while enforcing existing permissions, and users can share Quick applications as public websites.
