The all-MiniLM-L12-v2 model from Sentence Transformers is now available in Amazon SageMaker JumpStart, mapping sentences and paragraphs to a 384-dimensional dense vector space for semantic search, text clustering, and sentence similarity. Its compact architecture delivers fast inference while maintaining strong embedding quality, and it can be deployed with a few clicks via SageMaker Studio or the SageMaker Python SDK.
