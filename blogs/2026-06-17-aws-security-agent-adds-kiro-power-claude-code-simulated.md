AWS Security Agent (now part of AWS Continuum) adds support for Kiro and Claude Code to trigger security scans from the development environment, plus simulated validations that confirm code scanner findings by exploiting them in a sandbox and providing proof of exploit to minimize false positives. New integrations include GitLab.com, GitLab Self Managed, GitHub Enterprise, Bitbucket, and Confluence. Available in all regions where AWS Security Agent is supported.
