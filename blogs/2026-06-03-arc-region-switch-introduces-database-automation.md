Amazon Application Recovery Controller (ARC) Region switch introduces three new execution blocks automating database scaling and failover for multi-Region workloads. The blocks support Aurora serverless scaling, Aurora provisioned scaling, and Neptune global database failover with cross-account orchestration for comprehensive disaster recovery.
