Amazon Bedrock Guardrails now offers the InvokeGuardrailChecks API, a resourceless detect-only API that applies individual safeguards at any point in an agent loop without creating guardrail resources, returning numeric severity and confidence scores for custom thresholds. It supports content filters, prompt attack detection (jailbreak, prompt injection, prompt leakage as independent checks), and sensitive information filters. Available in US East (N. Virginia, Ohio), US West (Oregon), Europe (London, Stockholm), and Asia Pacific (Tokyo, Sydney).
