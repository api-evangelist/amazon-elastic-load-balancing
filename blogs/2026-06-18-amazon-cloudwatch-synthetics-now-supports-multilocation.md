Amazon CloudWatch Synthetics now lets you run the same canary across multiple AWS Regions from a single primary Region, which automatically replicates it and consolidates all run data, metrics, and artifacts. You can configure alarms that activate only when issues are detected from multiple locations, and existing single-region canaries can be upgraded without recreating them. Available in all commercial Regions that support CloudWatch Synthetics.
