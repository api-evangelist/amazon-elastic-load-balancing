AWS IoT Core provides two new CloudWatch Log event types: Ping log emitted for MQTT Keep-alive messages and Connection.AuthNError log recording rejected authentication attempts. These logs enable faster troubleshooting of device connectivity and credential issues for IoT fleet management.
