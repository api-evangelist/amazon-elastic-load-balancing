Amazon SageMaker Unified Studio now enables scheduling, parameterizing, and orchestrating notebook runs directly from the notebook interface. Features include on-demand background runs, scheduled recurring execution, notebook parameterization, and AI-assisted troubleshooting using Data Agent for streamlined data science workflows.
