Amazon EKS Capabilities can now be configured as log delivery sources using Amazon CloudWatch Vended Logs, enabling monitoring and troubleshooting of managed controllers for Argo CD, AWS Controllers for Kubernetes, and kro. Logs can be reliably delivered to CloudWatch Logs, Amazon S3, or Amazon Kinesis Data Firehose endpoints across all AWS Regions supporting EKS Capabilities.
