Amazon MQ for RabbitMQ is now available in the AWS European Sovereign Cloud (Germany) Region, a cloud located entirely within the EU designed to help customers in regulated industries and public sector organizations meet their sovereignty requirements. This expansion enables European customers to run managed message broker workloads while maintaining strict data residency controls.
