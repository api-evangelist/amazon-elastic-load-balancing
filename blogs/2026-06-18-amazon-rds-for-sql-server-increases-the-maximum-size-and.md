---
title: "Amazon RDS for SQL Server increases the maximum size and provisioned performance of General Purpose (gp3) volumes"
url: "https://aws.amazon.com/about-aws/whats-new/2026/06/rds-sqlserver-increases-gp3-limits/"
date: "2026-06-18"
feed_url: "https://aws.amazon.com/about-aws/whats-new/recent/feed/"
---
Amazon RDS for SQL Server now supports gp3 volumes up to 64 TiB (4X the prior 16 TiB), up to 80,000 IOPS (5X the prior 16,000), and up to 2,000 MiB/s throughput (2X the prior 1,000). Customers can also add up to three gp3 or io2 volumes per DB instance for up to 256 TiB total capacity, with no change to pricing.
