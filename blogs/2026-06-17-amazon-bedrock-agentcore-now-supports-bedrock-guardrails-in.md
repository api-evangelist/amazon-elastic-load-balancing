Amazon Bedrock AgentCore now supports Bedrock Guardrails in AgentCore policy, evaluating the outputs of every authorized agent action and inputs to every gateway target call in real time to detect and block prompt injection, harmful content, and sensitive information exposure. Evaluations occur at the gateway perimeter outside the agent's code and are logged via AgentCore observability. Available in US East (N. Virginia), Europe (London, Stockholm), and Asia Pacific (Sydney, Tokyo).
