AWS Deadline Cloud now supports persistent storage for Service-Managed Fleets, allowing you to maintain data across worker lifecycle events using configurable Amazon EBS volumes. Customers can set configurable time-to-live on persistent EBS volumes to balance storage costs with startup performance for rendering and content creation workloads.
