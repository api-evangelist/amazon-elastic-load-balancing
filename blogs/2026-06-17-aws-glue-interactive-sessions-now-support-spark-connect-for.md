AWS Glue Interactive Sessions now support Apache Spark Connect, letting you develop and run Spark applications from SageMaker Unified Studio notebooks, Jupyter, or VS Code on Glue's serverless infrastructure without managing clusters. A thin-client architecture decouples the client from the Spark execution environment, enabling ad hoc exploration, iterative debugging, and incremental PySpark development, with Spark UI, History Server, and session management. Available in 15 Regions.
