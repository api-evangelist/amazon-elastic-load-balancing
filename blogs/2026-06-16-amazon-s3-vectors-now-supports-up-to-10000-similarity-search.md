Amazon S3 Vectors can now return up to 10,000 similarity search results per query, a 100x increase from the previous limit, helping multi-stage retrieval pipelines apply reranking, aggregations, or deduplication. Specify up to 10,000 topK nearest neighbors in a QueryVectors API request, with results returned across multiple pages; the first 512 KB of data returned per query is free. Available in all Regions where S3 Vectors runs.
