AWS Config now supports internal service linked rules, enabling AWS services to evaluate AWS resource configurations using AWS Config managed rules. AWS Security Hub CSPM uses this capability to provide integrated security evaluations without charging customers for evaluations, reducing compliance monitoring costs.
