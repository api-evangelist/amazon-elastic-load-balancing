Amazon ElastiCache now supports durability enabling use for workloads that require microsecond read latency but cannot tolerate data loss. Options include synchronous writes for zero data loss and asynchronous writes maintaining microsecond latency, giving teams flexibility to balance performance with data persistence requirements.
