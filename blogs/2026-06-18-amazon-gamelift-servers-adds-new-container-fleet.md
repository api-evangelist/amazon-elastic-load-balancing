Amazon GameLift Servers now lets you customize Linux capabilities for containers (such as NET_RAW or SYS_PTRACE) in container group definitions, and adds a new ListContainersNetworkInfo() server SDK action that returns container name, ID, local IP, and group type for service discovery between game servers and auxiliary services. The action is supported in server SDK 5.x for Go, C++, and C#, plus Unreal Engine and Unity plugins, and the features are available in all GameLift Servers regions except China.
