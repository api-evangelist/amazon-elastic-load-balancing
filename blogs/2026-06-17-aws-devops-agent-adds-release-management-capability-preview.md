AWS DevOps Agent now offers a release management capability in preview that reviews code changes for release readiness and runs autonomous release testing. Release readiness review checks for drift from internal standards, dependency impacts, and access controls, and maps cross-repository dependencies, while release testing generates and runs test plans for web and API applications. Available in US East (N. Virginia) at no additional cost during preview.
