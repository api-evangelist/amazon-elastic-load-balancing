Oracle Database@AWS now supports Oracle Autonomous AI Database Serverless (ADB-S), a fully managed database on Exadata infrastructure that handles patching, tuning, and scaling, provisionable from the AWS console, CLI, or APIs. ADB-S supports four workload types (AI Transaction Processing, AI Lakehouse, AI JSON Database, Oracle APEX), includes Autonomous Data Guard, automated S3 backups, and cross-Region DR, and integrates with AWS KMS, CloudWatch, and EventBridge. Available in US East (N. Virginia) and US West (Oregon).
