Amazon RDS for SQL Server launches Bring Your Own Media (BYOM) for Microsoft SQL Server, enabling customers to reuse their existing Microsoft SQL Server licenses through License Mobility. Integration with AWS License Manager tracks usage for compliance, reducing software licensing costs for enterprise database workloads.
