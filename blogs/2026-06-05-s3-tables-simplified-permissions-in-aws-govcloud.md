AWS Glue Data Catalog now supports IAM-based authorization for Amazon S3 Tables and Apache Iceberg materialized views, available in AWS GovCloud (US-East) and (US-West) Regions. This capability enables defining permissions across storage, catalog, and query engines in a single IAM policy, simplifying access management for government and regulated workloads.
