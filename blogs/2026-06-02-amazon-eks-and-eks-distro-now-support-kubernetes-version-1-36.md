Amazon EKS and EKS Distro now support Kubernetes version 1.36, which brings User Namespaces to general availability for mapping container root to an unprivileged host user, and Mutating Admission Policies for CEL-based resource mutations. Additional improvements include In-Place Pod-Level Resources Vertical Scaling allowing pods to resize shared CPU and memory budget without restart.
