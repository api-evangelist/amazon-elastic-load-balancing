The AWS Model Context Protocol (MCP) Server now supports cross-account and cross-role access, allowing AI coding agents to work across multiple AWS accounts and IAM roles within a single session without restarts. Users can specify profiles per command for seamless switching between accounts, enabling more flexible multi-account development workflows.
