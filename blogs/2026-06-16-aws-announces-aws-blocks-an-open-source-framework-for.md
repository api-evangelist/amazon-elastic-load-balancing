AWS Blocks is a public-preview open-source TypeScript framework for building backend capabilities on AWS without learning infrastructure tools, running a fully functional local environment with Postgres, authentication, and real-time messaging with no AWS account required. The same application code deploys to production AWS services with zero changes, and developers can drop into AWS CDK at any point. At preview it supports SPAs like Vite + React and SSR frameworks such as Next.js, Nuxt, and Astro, available at no additional charge.
