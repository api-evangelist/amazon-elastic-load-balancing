Amazon SNS customers in the Asia Pacific (Seoul) Region can now send SMS text messages to subscribers in more than 200 countries and territories via AWS End User Messaging. Customers can subscribe phone numbers to SNS topics and broadcast SMS messages from the Seoul Region.
