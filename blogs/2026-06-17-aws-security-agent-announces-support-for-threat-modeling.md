AWS Security Agent (now part of AWS Continuum) now includes threat modeling in public preview, an agentic capability that analyzes design documents or source code to identify threats with recommended mitigations using the STRIDE framework across all six categories. Developers can integrate it into IDEs such as Kiro and Claude Code to create threat models from specs early in design. Available in all regions supported by AWS Security Agent at no additional cost during preview.
