AWS Config now supports 9 additional resource types across Amazon Bedrock, Bedrock AgentCore, and Amazon SageMaker. The newly supported types include Flow Aliases, Evaluators, Gateway Targets, Online Evaluation Configs, Runtime Endpoints, and SageMaker Clusters, Endpoints, Model Package Groups, and Pipelines for comprehensive infrastructure visibility.
