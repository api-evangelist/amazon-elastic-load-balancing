AWS now offers Ubuntu 24.04 LTS bundles for Amazon WorkSpaces Personal in the AWS China (Ningxia) Region, operated by NWCD, bringing updated Linux packages, toolchains, security improvements, and an extended support lifecycle. The bundle provides a migration path for customers running older Amazon Linux 2 WorkSpaces and is selectable when creating a new Linux WorkSpace.
