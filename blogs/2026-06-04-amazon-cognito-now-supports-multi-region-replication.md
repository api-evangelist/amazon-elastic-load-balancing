Amazon Cognito now supports multi-Region replication, synchronizing user and machine identity data to a secondary user pool in near real-time. During regional disruptions, traffic can redirect to the secondary pool while signed-in users continue accessing their applications without re-authenticating, supporting all authentication methods including federated sign-in.
